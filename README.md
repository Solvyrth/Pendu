# Jeu du Pendu 🎯

Un jeu du pendu classique en français avec interface graphique développé en Python.

## Fonctionnalités ✨

- Interface graphique intuitive avec Tkinter
- Construction progressive de la potence et du pendu
- Dictionnaire français intégré (sans accents)
- Proposition de lettres ou de mots complets
- Tableau de score (Joueur vs Ordinateur)
- Gestion des lettres accentuées (é → e, à → a, etc.)

## Installation 🚀

1. Clonez ou téléchargez ce projet
2. Placez un fichier `mots_francais.txt` (liste de mots français, un par ligne) dans le dossier
3. Lancez le jeu :
   ```bash
   python3 Pendu.py
   ```

## Comment jouer 🎮

- Proposez une lettre ou tentez le mot complet
- Appuyez sur Entrée ou cliquez sur "Proposer"
- 10 erreurs maximum avant que le pendu soit complet
- Le score se met à jour automatiquement
